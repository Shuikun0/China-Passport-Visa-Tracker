# 中国护照签证便利度追踪

一个干净、纯前端、无后端的签证可视化小工具。  
给中国护照出行做快速判断：一眼看懂，少查半天。

## 这个工具解决什么

- 哪些国家免签 / 落地签？
- 哪些是电子签，哪些是常规签？
- 哪些国家“办签证相对容易”？

## 功能

- 世界地图交互展示（签证方式 / 签证容易度双模式）
- 国家列表搜索、筛选、排序
- 详情面板（备注 + 实用链接）
- 简约日间 / 夜间主题

## 使用方式

直接用浏览器打开 `index.html` 即可。

不需要构建，不需要服务端，不折腾。

## 数据维护

在 `index.html` 里直接维护：

- `STATIC_VISA_DATA`
- `POPULAR_VISA_LINKS_BY_ISO3`

## 提醒

签证政策变化很快。  
本项目适合作为“快速参考层”，实际出行请以官方渠道为准。
